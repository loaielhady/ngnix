>#### nginx ("engine x") is an open source HTTP web server, reverse proxy, content cache, load balancer, TCP/UDP proxy server,and mail proxy server.

# install in different Operating system

1. üêß On Debian/Ubuntu-based systems:

```
sudo apt update
sudo apt install nginx
sudo systemctl start nginx
sudo systemctl enable nginx
   ```
2. üêß On CentOS/RHEL-based systems:

```
sudo yum install epel-release
sudo yum install nginx
sudo systemctl start nginx
sudo systemctl enable nginx

```

3. Verification:
After installation, verify Nginx is running by opening a web browser and navigating to `http://your_server_ip_address` or `http://localhost` You should see the default Nginx welcome page.

4. Configuration:
Nginx configuration files are typically located in `/etc/nginx/nginx.conf` and can be modified to suit specific needs. After making changes to the configuration, test the configuration and reload Nginx:

```
sudo nginx -t
sudo systemctl reload nginx
```
